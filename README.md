# ENISA SRP Demo v2

Static prototype for the ENISA SRP readiness demo system, generated from PRD v2.

This version is preserved as the previous design-system iteration for comparison against v3.

## View Locally

```bash
python3 -m http.server 8080
```

Open:

```text
http://127.0.0.1:8080/index.html
```
